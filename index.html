Sobre Nosotros

Somos un colectivo de desarrolladores de software, cursando el séptimo semestre de Ingeniería, unidos por la pasión de emprender y aplicar nuestro conocimiento académico en soluciones para el mundo real.

Nuestra formación nos ha proporcionado un dominio sólido en el desarrollo full-stack. Creamos interfaces de usuario dinámicas y atractivas con HTML, CSS y JavaScript, y construimos la lógica de negocio robusta y escalable que las respalda utilizando Java y Python.

Nos encontramos en una etapa emocionante, buscando activamente proyectos que nos permitan no solo expandir nuestro portafolio, sino también aportar una perspectiva fresca y soluciones innovadoras a nuestros clientes. Estamos comprometidos con la excelencia y listos para convertir conceptos creativos en productos digitales funcionales y de alta calidad.

Nuestros Proyectos

PEEK

Una página web dónde los usuarios podría adoptar mascotas y además publicar perdidas de mascotas en sus coloneas, donde se ocupa backend y frontend con Visual Studio, HTML y CSS

Huellas & Estilo

Aplicación movil donde se crean citas de baños y cortes a mascotas, para poder ayudar a la estetica y tener un mejor cotrol de sus clientes, donde se utilizas AndroidStudio para crearlo con Firebase.

Componentes MX

Una aplicación de escritorio para ayudar al cliente a contabilizar sus ventas de componentes de PC's, donde se ocupó el lenguaje de programación de Visual Code Y MVC.
